# 수정 내용 및 추가된 기능

## 사용 방법
1. 첨부 파일 `FolderWorkBook.html` 더블 클릭
2. 워크북 이미지 폴더를 브라우저 화면에 드래그 앤 드롭 또는 "폴더 선택하기" 버튼 클릭

**※ 참고: 웹 브라우저 권한 이슈로 폴더 드래그 앤 드롭은 1번 방법(로컬 파일 더블클릭)으로 실행한 경우에만 가능합니다.**
